# Lab - SPF 100 Introduction

## Lab Objectives
In this lab, we will introduce you to the Security Professionals Friend (SPF 100). You will learn to accomplish the following tasks.

 - Familiarizing yourself with our project
 - Executing Python modules vs Python programs
 - Executing Programs in a virtual environment with elevated privileges
 - Installing modules with pip
 - Installing modules in "development mode" with pip
 - Using entry points to create Operating System executables 


## Instructions

This lab has multiple steps. To complete this lab, you will have to closely follow the instructions in the workbook. Once you have finished the lab, you will submit an updated module installer that will create executable programs in the Operating System's path.
